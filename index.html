Ejercicio 1

Introduce los datos académicos de un grupo de estudiantes (Nombre, Materia, Notas) y genera un listado ordenado por promedio. El maestro debe poder seleccionar una materia y ver solo las notas de los alumnos en esa asignatura. No se permite el uso de bases de datos ni archivos externos.

Ejercicio 2

Genera una matriz con valores ingresados por el usuario, rellenando el resto con números aleatorios. El usuario debe poder seleccionar una fila y una columna, y el sistema mostrará el valor en esa posición.

Ejercicio 3

Crea un programa que sume dos matrices, independientemente de sus dimensiones.

Ejercicio 4

Desarrolla un programa que calcule el producto de varias matrices. Todas deben tener el mismo número de filas y columnas. El usuario debe decidir cuántas matrices quiere multiplicar.

Ejercicio 5

Desarrollar un codigo que permita al usuario generar una matriz de un numero de columnas y filas determinadas por el usuario , al generarlas debe permitir al usuario ingresar valores y eliminar los valores repetidos , conservando solo la primera aparicion del valor

Ejercicio 6

Permitir al usuario generar un vector de x magnitudes y formar a partir de esto un arreglo de valores primos y otro de valores compuestos

Ejercicio 1 leet code

Create Hello World Function

Ejercicio 2 leet code

Given an integer n, return a counter function. This counter function initially returns n and then returns 1 more than the previous value every subsequent time it is called (n, n + 1, n + 2, etc).

Ejercicio 3 leet code

Write a function expect that helps developers test their code. It should take in any value val and return an object with the following two functions. toBe(val) accepts another value and returns true if the two values === each other. If they are not equal, it should throw an error "Not Equal". notToBe(val) accepts another value and returns true if the two values !== each other. If they are equal, it should throw an error "Equal".

Ejercicio 4 leet code

Write a function createCounter. It should accept an initial integer init. It should return an object with three functions. The three functions are: increment() increases the current value by 1 and then returns it. decrement() reduces the current value by 1 and then returns it. reset() sets the current value to init and then returns it.

Ejercicio 5 leet code

Given an integer array arr and a mapping function fn, return a new array with a transformation applied to each element. The returned array should be created such that returnedArray[i] = fn(arr[i], i). Please solve it without the built-in Array.map method.

Ejercicio 6 leet code

Given an integer array arr and a filtering function fn, return a filtered array filteredArr. The fn function takes one or two arguments: arr[i] - number from the arr i - index of arr[i] filteredArr should only contain the elements from the arr for which the expression fn(arr[i], i) evaluates to a truthy value. A truthy value is a value where Boolean(value) returns true. Please solve it without the built-in Array.filter method.
